# Control Client Admin Query Methods

## Goal

Expose Rust `HttpControlClient` helpers for Control admin paginated list
queries so CLI/admin SDK code can reuse the same pagination, filtering, and
sorting contract as the web console.

## Scope

- [x] Add tests for supported query path encoding.
- [x] Add compile-time coverage for paginated admin list query methods.
- [x] Add `_with_query(AdminListQuery)` helpers for:
      users, audit logs, usage summaries, admin sessions, controllers,
      controlled devices, plan catalog, relay credentials, relays, and device
      access grants.
- [x] Percent-encode query string values generated by the Rust client.
- [x] Document the Rust client workflow in `README.md`.
